# Installation

## npm
```$ yarn add react-query```

## Devtools

```$ yarn add react-query-devtools```


```
import { ReactQueryDevtools } from 'react-query-devtools'
 
 function App() {
   return (
     <>
       {/* The rest of your application */}
       <ReactQueryDevtools initialIsOpen />
     </>
   )
 }
```
